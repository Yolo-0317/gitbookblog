#### 清除浮动的方式

- 添加额外标签

```html
<div class="parent">
  //添加额外标签并且添加clear属性
  <div style="clear:both"></div>
  //也可以加一个br标签
</div>
```

- 父级添加 overflow 属性，或者设置高度

- 建立伪类选择器清除浮动

```css
/* 在css中添加:after伪元素 */
.parent:after{
  /* 设置添加子元素的内容是空 */
  content: '';
  /* 设置添加子元素为块级元素 */
  display: block;
  /* 设置添加的子元素的高度0 */
  height: 0;
  /* 设置添加子元素看不见 */
  visibility: hidden;
  /* 设置clear：both */
  clear: both;
}
```
